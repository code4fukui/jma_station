# 気象庁 地点番号オープンデータ

気象庁が提供するアメダス地点情報履歴ファイルをCSV/JSONデータ化したリポジトリです。

## デモ
[気象庁 地点番号マップ](https://code4fukui.github.io/jma_station/)

## 機能
- 気象庁の地点番号一覧データを CSV/JSON形式で提供
- 終了した地点番号も含む完全なデータセットと、現在運用中の地点番号のみのデータセットを用意
- 自動更新スクリプトを含む

## 使い方
国内の気象観測地点に関する情報を自由に活用できます。

## データ・API
- [地点番号データ](https://code4fukui.github.io/jma_station/jma_station.csv)
- [アクティブな地点番号データ](https://code4fukui.github.io/jma_station/jma_station_active.csv)
